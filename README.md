# hello_world

> Practice to create a simple C project and setup the CMakeList.txt file.

## structure

~~~
/home/terry/Documents/github/hello_world
├── build
├── CMakeLists.txt
├── inc
│   └── foo.h
├── lib
├── README.md
└── src
    ├── foo.c
    └── main.c

4 directories, 5 files
~~~

## implement commands

1. mkdir build
2. cd build
3. cmake ../
4. make
5. cmake --install . --prefix "/path/to/install/folder"
